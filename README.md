# Investigación Área Foliar

## Objetivo

Crear un algoritmo capaz de identificar el porcentaje de nutrientes en una hoja basado en su color

## Información
Se cuenta con 64 muestras de hojas de aguacate Hass del Estado de Nayarit.

## Nutrientes
* N
* P
* K
* Ca
* Mg
* S
* Fe
* Cu
* Mn
* Zn
* B
* Mo
* Na
* Cl
